# 문제은행 (QUIZBANK_WEB)

![메인](https://github.com/user-attachments/assets/8674c60b-20cf-4792-afc1-7f213a7fc610)


## 프로젝트 소개
1학년 2학기 프로그래밍언어실습, 스크립트언어실습의 기말고사 프로젝트로, 1학기에 문제은행 콘솔 모드에서 더욱 기능적으로 발전한 WEB으로 구현한 프로젝트이다.
사용자는 문제를 등록할 수 있는 문제 게시판 커뮤니티에 시험 보기 기능 및 이벤트를 추가하여 현실성있는 웹사이트를 구현함을 목표로 하였다.

## 개발 기간
2024-11-20 ~ 25-01-05

## 개발자 소개
한국폴리텍대학 인공지능소프트웨어과 1학년 학생 - 본인 (1명)

## 개발 환경
Ecplise, Apache Tomcat 10.1, jdk-17, sqldeveloper

## 기술 스택

### 프론트 엔드
HTML, JS, CSS

### 백 엔드
JAVA, JSP

### 데이터 베이스
Oracle Database
